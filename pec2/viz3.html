PEC 2 - Visualización 3: Bullet graph

Descripción

Esta visualización muestra un bullet graph que representa el dinero obtenido en ventas por cada tienda de una cadena de supermercados, con el objetivo de comparar el rendimiento de cada tienda en relación con su objetivo de ventas establecido por la cadena.

En el dataset utilizado, el cual está disponible aquí, contiene 1.980 registros de una cadena de supermercados con múltiples establecimientos, y recoge datos detallados de transacciones en diversas tiendas, categorías de productos, interacciones con los clientes y actividades promocionales. El conjunto de datos abarca aproximadamente dos años de datos de ventas (2023-2025) y ofrece información sobre el comportamiento de compra de los clientes, el rendimiento de las tiendas, la popularidad de los productos y la eficacia de las promociones.

Comentar que se ha realizado un preprocesamiento de los datos para agregar las ventas por tienda y calcular el atributo del objetivo de ventas de cada una de forma aleatoria, con poder ilustrar la utilidad de un bullet graph en este contexto.

Este diagrama se ha desarrollado utilizando Tableau Public.
